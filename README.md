# Code the Collection: Two-day open cultural data hackathon
A two-day open cultural data hackathon hosted by the Centre for Digital Scholarship (Bodleian Libraries) and the Stabi Lab (Berlin State Library). Create your own digital projects based on open data and curated datasets.

## Overview & objectives
 - Raise awareness of the various cultural datasets that can be accessed, especially open data and curated datasets of the Stiftung Preußischer Kulturbesitz and the Libraries and Museums at the University of Oxford.
 - Raise awareness of the range and opportunities of GLAM data.
 - To apply digital approaches to GLAM data.
 - To understand the value of GLAM datasets beyond the library and to understand the ways these datasets are used by researchers/students/readers, etc.

## Purpose of this Repo
This repo presents a set of six project proposals developed for the Code the Collection hackathon. Working from ten open cultural heritage datasets spanning museum collections, medieval manuscripts, WW1 archives, historical audio recordings, typographic specimen books, and provenance records, I analysed what each dataset contained and identified the core problems that currently make these collections difficult to access, explore, or connect with one another.

From that analysis, I proposed six buildable digital systems — ranging from beginner to advanced — each designed to solve a specific gap. These include an interactive geographic map linking artefacts across Oxford and Berlin, a provenance network visualiser for colonial-era acquisitions, an NLP-powered WW1 archive search tool, a computer vision classifier for historical typefaces, a data-driven story of European dining culture through 19th-century menu cards, and a no-code query interface for the Bodleian's medieval manuscript catalogue. Each proposal maps directly to real datasets, recommended tools, and a realistic two-day build plan — making them ready to pitch and prototype at the hackathon.
